# dbfp7-adhesion-analysis
Code supporting the publication Obille et al., 2026. This repository contains AFM data processing scripts and analysis workflows used to generate figures and quantitative results in the article.

## Contents
- `dbfp7-adhesion-analaysis.R`: Main analysis script

Raw AFM image files are available from the corresponding author upon reasonable request.

## Dependencies
R version 4.4.0
Packages: tidyr, dplyr, ggplot2, stringr, tibble, tidyverse
